# Proof-of-Workforce

A modern, dark-themed React SaaS demo for AI-powered candidate verification with blockchain-style proof generation.

## Core Idea

Proof-of-Workforce adds trust to hiring:
- AI evaluates candidate profiles and produces score/summary insights.
- Blockchain module generates a tamper-proof style hash for verification.
- Recruiters manage and filter candidate trust status from a dashboard.

## Tech Stack

- React + Vite
- Tailwind CSS (Vite plugin)
- Framer Motion
- React Router
- React Hot Toast
- Local state + `localStorage` persistence

## Pages

- Landing page with animated hero and feature cards
- Candidate Analysis form with simulated AI processing and skeleton loading
- AI Report view with score visualization and blockchain proof generation
- Recruiter Dashboard with filters and modal report viewer
- Verification page for hash-based proof lookup

## Run Locally

```bash
npm install
npm run dev
```
